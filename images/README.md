# Images

This directory contains images for documentation and project showcase.

## Directory Structure

```
images/
├── robot/              # Photos of the assembled robot
├── circuit/            # Circuit diagrams and schematics
├── track/              # Test track examples
├── assembly/           # Step-by-step assembly photos
└── results/            # Robot in action, demonstrations
```

## Adding Images

When adding images to this repository:

1. **Use descriptive filenames**
   - Good: `assembled-robot-front-view.jpg`
   - Bad: `IMG_1234.jpg`

2. **Optimize file sizes**
   - Compress images before uploading
   - Keep individual files under 1MB
   - Use JPEG for photos, PNG for diagrams

3. **Recommended dimensions**
   - Documentation images: 1200px max width
   - Thumbnail images: 400px max width
   - High-res for details: 2000px max width

4. **Image types needed**
   - Assembled robot (multiple angles)
   - Circuit connections close-ups
   - LCD display showing different states
   - Track layout examples
   - Component placement
   - Wiring diagrams (photographed or drawn)

## How to Reference Images in README

Use relative paths in markdown:

```markdown
![Robot Front View](images/robot/front-view.jpg)
![Circuit Diagram](images/circuit/wiring-diagram.png)
```
